Tåîchô (sometimes called "Dogrib" in informal settings) is the Winmac Dene Font (before Unicode) way of writing Tłı̨chǫ. This font hack was developed by Jim Stauffer of Whatì.

It doesn't really matter if you prefer to create documents using Winmac encoding or the newer Unicode system, but make sure that things that are distributed on the web or in databases are properly managed.
